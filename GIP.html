GIP

Wat is mijn GIP

Mijn GIP is een zelfgebouwd autootje die je kan besturen met je smartphone, in de auto zit een Arduino verwerkt.

Wireframes website

Logboek

Toen meneer Lippens ons de opdracht gaf om 5 projecten te zoeken die te maken hebben met physical computing waren dit mijn ideeën. Jammer genoeg had ik geen 5 ideeën.

  • Een auto die je kan besturen via je smartphone.

  • Een arm die je kan besturen via knoppen.

  • Een robot spin die kan bewegen.

Ook heb ik alle dingen die ik nodig had opgezocht. Ik zou mijn spullen bij Ali-Express kopen dit was een riskante keuze omdat ze er om bekend staan dat de leveringen zeer lang duren.

In de kerstvakantie ben ik begonnen met deze website. Allereerst heb ik de wireframes gemaakt, nu ik naar mijn website kijk vind ik dat hij er best goed op lijkt.
